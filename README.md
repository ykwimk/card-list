# React Card List

## 📂 실행 방법
```
  1. yarn

  2. yarn start
```

<br />

## 📂 구조

```
├── components
|      ├── Card
|      ├── Header
|      ├── ProductList
|      └── WishList
|
├── components
|      └── ListContainer
|
├── stores
|      └── list.js
|
├── App.js
|
├── index.js
|
├── package.json
|
└── README.md
```

<br />

## 📂 설명

- `React.js`, `Mobx`, `Scss`, `classNames`, `axios` 를 사용.
- 주요 기능인 Infinite Scroll과 Lazy Image Loading은 `IntersectionObserver API` 를 사용.
